# Документація до лабораторної роботи №6
## Тема: Системи обробки відеопотоку в реальному часі на базі OpenCV

### 1. Огляд проєкту
Дане програмне забезпечення призначене для захоплення відеоданих з пристрою введення та їх трансформації за допомогою алгоритмів комп'ютерного зору. Проєкт реалізує обробку кадрів у реальному часі з можливістю динамічного перемикання режимів фільтрації через інтерфейс клавіатури.

### 2. Функціональний склад
Система забезпечує виконання наступних завдань:
* Ініціалізація та стабільне зчитування відеопотоку з камери.
* Візуалізація оброблених кадрів у графічному вікні.
* Обробка подій клавіатури для зміни параметрів візуалізації.

### 3. Модульна архітектура
Проєкт побудований на принципах поділу відповідальності (Separation of Concerns):

| Модуль | Функціональне призначення |
| :--- | :--- |
| **CameraProvider** | Керування доступом до апаратного забезпечення та захоплення кадрів. |
| **KeyProcessor** | Логіка інтерпретації вводу та управління станами програми. |
| **FrameProcessor** | Реалізація математичних алгоритмів фільтрації зображення. |
| **Display** | Модуль рендерингу графічного інтерфейсу та службового тексту. |

### 4. Технічні характеристики
* **Мова:** C++17
* **Бібліотеки:** OpenCV (core, imgproc, highgui)
* **Система збірки:** CMake 3.10+
* **Цільова ОС:** Linux (Ubuntu/Kali)

### 5. Інструкція з експлуатації

#### 5.1. Попередня настройка
Встановлення необхідних системних залежностей:
```bash

./preinstall.sh


chmod +x build.sh
./build.sh


chmod +x ./run.sh
```

### 6. Управління та режими обробки
Зміна алгоритмів обробки здійснюється в реальному часі під час виконання програми за допомогою наступних гарячих клавіш:

#### 6.1. Режими фільтрації зображення
* `n` — **Normal**: Повернення до вихідного відеопотоку без накладання графічних ефектів.
* `c` — **Canny**: Активація детектора меж Кенні для виділення контурів об'єктів.
* `b` — **Blur**: Застосування фільтра Гауса для пом'якшення (розмиття) зображення.
* `i` — **Invert**: Попіксельна інверсія колірних каналів (ефект негативу).

#### 6.2. Керування системними процесами
* `q` / `ESC` — Коректне завершення роботи програми, зупинка захоплення відео та звільнення виділених ресурсів пам'яті.
