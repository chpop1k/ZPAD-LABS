# Документація до лабораторної роботи №7
## Тема: Багатопотокова детекція об'єктів за допомогою нейронних мереж та OpenCV

### 1. Огляд проєкту
Дана робота присвячена реалізації асинхронної системи комп'ютерного зору. Програма виконує захоплення відеопотоку, його паралельну обробку нейронною мережею (SSD на базі ResNet-10) та застосування стандартних фільтрів трансформації зображення.

### 2. Ключові особливості
* **Асинхронна архітектура**: Обробка кадрів нейромережею винесена в окремий потік (background thread), що дозволяє підтримувати високий FPS основного відеопотоку.
* **Синхронізація даних**: Безпечна передача кадрів та результатів детекції між потоками за допомогою м'ютексів (`std::mutex`) та атомарних змінних.
* **DNN Модуль**: Використання глибоких нейронних мереж для ідентифікації облич у реальному часі.

### 3. Компоненти системи
Проєкт розділений на функціональні модулі для забезпечення масштабованості:

| Файл | Опис компонента |
| :--- | :--- |
| **CameraProvider** | Модуль ініціалізації відеопристрою та захоплення сирих матриць даних. |
| **FaceDetector** | Воркер-клас, що керує життєвим циклом фонового потоку та нейромережею. |
| **FrameProcessor** | Набір алгоритмів для попіксельної обробки зображення (Canny, Gaussian Blur, Invert). |
| **KeyProcessor** | Аналізатор вводу, що відповідає за диспетчеризацію режимів роботи. |

### 4. Технологічний стек
* **Мова програмування:** C++17
* **Бібліотека КЗ:** OpenCV 4.x (модулі core, imgproc, highgui, dnn)
* **Система збірки:** CMake 3.10+
* **Модель детекції:** Single Shot Multibox Detector (SSD) Caffe model

### 5. Інструкція зі збірки та запуску

#### 5.1. Завантаження ресурсів
Перед початком необхідно завантажити конфігураційні файли та ваги нейронної мережі:
```bash
chmod +x preinstall.sh
./preinstall.sh

chmod +x build.sh
./build.sh

```
## 6 Управління режимами

Програма підтримує зміну режимів обробки зображення за допомогою клавіатури.

###  Клавіші керування

| Клавіша | Режим | Опис |
|--------|------|------|
| `f` | Face Detection | Активація нейромережевого детектора облич |
| `c` | Canny | Виділення контурів об'єктів |
| `b` | Blur | Розмиття зображення за допомогою Gaussian Blur |
| `i` | Invert | Інверсія кольорів (негатив) |
| `n` | Normal | Відображення оригінального відеопотоку |
| `ESC` / `q` | Exit | Завершення роботи програми |

---

###  Примітки

- Перемикання режимів відбувається миттєво під час роботи програми.
- Обробка застосовується до кожного кадру відеопотоку в реальному часі.
- Для коректної роботи Face Detection переконайтесь, що відповідні моделі підключені.
